Baserom Credits

This baserom comes with a "Credits.txt" file that credits all of the resources used in the baserom and links to them. This is provided for your convenience and to serve as an example of how to do credits.

This project has no license nor do the maintainers claim any rights to external resources included in this project, those remain the rights of their respective authors.

It is good practice to keep track of resources you use in your project as you work on your hack, both so you know what you've included and as a helpful to way to credit the authors of resources when you go to release your hack. Especially if you intend to submit it to SMW Central which requires some form of credits alongside submissions (if not included in them).

Crediting this Baserom

All that is sufficient to credit this project in a hack submission to SMW Central is to include the provided Credits.txt file long with your hacks files, and remember to mention in any other credits you may create that it was built on the baserom.
